# ReactNative-instagram

## 프로젝트 개요
 1. 프로젝트명: 인스타 클론 프로젝트
 2. 참가자: 엄문주
 3. stack: React-Native, Expo, Firebase, 


## 프로젝트 목표
  1. React-Native를 이용해서 앱을 개발
  2. Redux를 이용해서 상태 관리
  3. Firebase를 이용해서 백엔드 없이 자체적으로 앱개발
  
## 프로젝트 구현 기능
  1. 회원가입, 로그인, 로그아웃
  2. 사진촬영, 사진업로드
  3. 사용자 검색
